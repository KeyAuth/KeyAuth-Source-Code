Bot Privacy Policy

We collect the following data

  • Seller key, a credential used on our API, along with the server ID the seller key was specified in

  • Command paramaters stored along with server ID if you've already done the /add command once. This makes it so you don't need to specify paramaters each time

  • Discord webhook links, if you specify one to the bot
